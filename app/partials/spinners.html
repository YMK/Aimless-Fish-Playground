CSS Based Spinners


These are some fully CSS based spinners. No JavaScript is needed to run them. They were created using box-shadows and animating them over different places. They should be really easy to integrate into your project.


Usage:

To add them to your site, create an element with 2 classes. The first is spinner and the second spinnerX where X is the number of spinner (shown on the right hand side. Then create one empty div inside.


<div class="spinner spinner1 mycustomclassforfunsies">
  <div></div>
</div>

Finally, to set the size of the spinner, set the font-size (I can explain why). The spinner will always be in a square bounding box the size of whatever you set the font size to be.


.mycustomclassforfunsies {
  font-size: 100px;
}

You can either use the file as a plain CSS file and include it in your project, or it also comes as a less file which can be used to modify the colours. To set all of the circles to be the same colour, use the less variable @onecolour, otherwise set the variables @colour1 to @colour6.
